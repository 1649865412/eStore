This is where we should put our published JSP template files. 
Note that JSP is the default and recommended templating technology, and the default directory is pages. 
Velocity and FreeMaker are other 2 popular templating technologies. However, to switch to use them, a few configurations are required.
Also, this directory can be configured in jspViewResolver or velocityConfig, they can be found in action-servlet.xml under WEB-INF.
